# bbm467-lyric-scraper

npm install

npm run dev

Usage:

Full relative path on csv file. e.g. ./example.csv
                                     ./my_files/example.csv

Make sure csv file has "artists" and "track_name" columns.
This will create a folder named /lyric_files with lyrics in it.
I shared my Genius token with you, if you have any errors, check for ".env" file and renew the token, client id, secret etc.
